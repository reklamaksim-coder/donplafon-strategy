3. Стратегические рычаги
Цель: Определить два главных рычага роста и их вклад в удвоение выручки.
Рычаг 1 — рост базы

От 1000 до ~1500 активных партнёров. Откуда +500?

  • Реактивация спящих (5000 база) → 180-220 партнёров
  • Реферальная сеть L2 → 100-130 партнёров
  • Прямые продажи Premium/HoReCa → 80-110 партнёров
  • Лендинг + контент → 50-70 партнёров
  • Школы/ивенты → 30-50 партнёров
Рычаг 2 — рост выручки на партнёра

+33% к среднему чеку через кросс-сейл

  • Электрокарнизы — допродажа существующим партнёрам
  • Умный дом — для Premium и HoReCa
Преимущество
Категории уже в ассортименте — продавать можно сегодня
Соотношение рычагов
РычагВкладСкоростьЗависимость от продукта
Рост базы (+500)~50%Медленнее (8 мес)Высокая (нужен кабинет, лендинг)
Рост выручки/партнёр (+33%)~50%Быстрее (можно начать сразу)Низкая (категории готовы)
Стратегическая логика
Кросс-сейл текущим партнёрам стартует немедленно. Рост базы блокирован продуктом — раздел 4.

13. KPI и система контроля
Цель: Измерять прогресс и вовремя корректировать курс.
Еженедельные метрики
  • Новые партнёры (регистрации + активации)
  • GMV за неделю
  • Количество заказов
  • Средний чек
Ежемесячные метрики
  • ARPPU по сегментам
  • Retention (% партнёров с повторным заказом)
  • Доля заказов с кросс-сейлом (карнизы, умный дом)
  • CAC по каналам
  • NPS партнёров
Квартальные ревью
  • Рычаг 1 vs рычаг 2: план-факт
  • Каналы привлечения: ROI по каждому
  • Сегментная структура: соответствует целевой?
  • Ревью рисков: какие сработали, какие новые
Алерты

При каких значениях бить тревогу:

  • SLA выше порога — сигнал о коллапсе сервиса
  • Retention ниже порога — отток
  • CAC растёт — канал перегревается
  • Доля кросс-сейла падает — менеджеры не продают
